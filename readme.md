# Seminario OOP/Ing-SW su LLM

Seminario Programmazione a Oggetti e Ingegneria del Software sui Large Language Model (LLM).

## In locale

```bash
# Clonare questo repo
git clone https://github.com/DCALab-UNIPV/seminario-oop-ing-sw-llms
cd seminario-oop-ing-sw-llms
python -m venv .env/ # Per creare un ambiente virtuale
source .env/bin/activate # Per entrare nell'ambiente
pip install -r requirements.txt # Per installare dipendenze
python src/silence.py # Per eseguire un file
```

Eseguire gli esempi in [src/](./src/).

## Su Colab

- Scaricare il notebook (file [`index.ipynb`](./index.ipynb)) dal repo:
